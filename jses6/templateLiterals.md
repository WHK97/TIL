# Template literals (JS문자다루기)
문자를 만들때 ""가아닌 ``(backtick)을 이용해 만들 수 있다
```
let String = `문자`;
```
백틱의 장점
1. 문자 중간 엔터키 사용이 가능하다.
```
let String = `문
자`;
```
2. 문자 중간에 변수를 넣을 때 편리하다.
JS에서 문자 중간에 변수를 넣고 싶을떄는 문자를 쪼개서 + 기호를 사용해야한다. 백틱을 사용할 경우 문자중간에 ${변수명}을 사용하여 넣을 수 있다.
```
let num = "숫자"
let String = `문자${num}`;
```
