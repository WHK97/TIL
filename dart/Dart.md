# Dart 기초 문법
## Dart 란?
https://dart.dev/

Dart는 모든 플렛폼에서 빠른 앱을 위한 클라이언트 최적화 언어이다
1. UI최적화
사용자 인터페이스 생성 요구에 특화된 프로그래밍 언어로 개발
2. 생산적인 개발
핫 리로드를 사용하여 실행 중인 앱에서 즉시 결과 확인이 가능하다.
3. 모든 플랫폼에서 빠르다
모바일, 데스크톱 및 백엔드용 ARM및 x64머신 코드로 컴파일한다. 웹은 JavaScript로 컴파일

Dart의 컴파일러 기술을 사용하면 다양한 방식으로 코드를 실행할 수 있다
1. 기본 플랫폼: 모바일 및 데스크톱 장치를 대상으로 하는 앱의 경우 Dart에는 Just-In-Time 컴파일 기능이 있는 Dart VM과 기계코드생성을 위한 Ahead-of-Time컴파일러가 모두 있다.
2. 웹 플랫폼: 웹을 대상으로 하는 앱의 경우 Dart는 개발 또는 프로덕션 목적으로 컴파일할 수 있다. 웹 컴파일러는 Dart를 JavaScript로 변환한다.
