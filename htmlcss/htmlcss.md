# HTML CSS

## HTML
HTML이란 하이퍼텍스트 마크업언어이다.
HTML은 태그를 사용한다.
[MDN](https://developer.mozilla.org/en-US/)
태그는 Block, inline태그가 있다 
- Block: 페이지의 가로 전체를 차지 하는 요소 (div, h1, p)
- Inline: 자신의 크기만큼만 차지하는 요소 (span, input, a)
종속태그 
다른 태그와 함꼐 상호작용을 해야 작동하는 태그를 말한다.
```
    <ul>
        <li>a</li>
    </ul>
    <ol>
        <li>a</li>
    </ol>
```
